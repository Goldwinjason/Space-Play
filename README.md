# Space Play

Visit the project:https://goldwinjason.github.io/Spaceplay/](https://goldwinjason.github.io/Space-Play/

## Overview

Space Play is an interactive space exploration and learning platform designed to make astronomy education engaging, immersive, and accessible. The platform enables students, educators, and space enthusiasts to explore planets, moons, and celestial bodies through interactive 3D visualizations and Web-based Augmented Reality (WebAR).

Built using modern web technologies, Space Play transforms traditional space education into a hands-on learning experience. Users can navigate through the Solar System, access educational content about planets and moons, explore planetary surfaces, and visualize celestial objects in Augmented Reality directly from their web browser.

The project aims to bridge the gap between theoretical learning and immersive visualization by making space science more interactive, engaging, and understandable.

## Project Architecture

Space Play consists of the following major modules:

### 1. Solar System Explorer

An interactive navigation system that allows users to travel through the Solar System and explore planets, moons, and celestial bodies.

### 2. Planet Information Module

Provides educational content, scientific facts, and descriptions about each planet and its characteristics.

### 3. Surface Exploration Module

Allows users to virtually explore the surface environment of selected planets and celestial objects.

### 4. WebAR Experience

Enables users to experience celestial objects through Augmented Reality, creating an immersive learning environment.

## Installation

### Clone the Repository

```bash
git clone https://github.com/Goldwinjason/Spaceplay.git
```

### Navigate to the Project Directory

```bash
cd Spaceplay
```

### Run the Project

Open the `index.html` file directly in a web browser.

Alternatively, use a local development server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
